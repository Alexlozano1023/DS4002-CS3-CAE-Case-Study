# Supplemental Materials for CAE Turbulence Case Study

This folder contains two readings that support the case study **Reconstructing Turbulence Fields Using Convolutional Autoencoders**. These materials provide the conceptual and technical background needed to understand how autoencoders work and why they are useful for reconstructing turbulence data.

---

## 1. Conceptual Explainer (Beginner-Friendly)
**Title:** An Introduction to Autoencoders (Analytics Vidhya)  
**Link:** https://www.analyticsvidhya.com/blog/2021/10/an-introduction-to-autoencoders/

This is a blog-style, accessible introduction to autoencoders.  
It explains:
- What autoencoders are  
- How they compress and reconstruct data  
- Why they are useful in practical applications  

**When to read it:**  
Before starting the notebook.  
This resource builds intuition and prepares you for the hands-on part of the case study.
